Техническое задание для разработки фитнес приложения

Общие сведения  
Цель разработки фитнес приложения - предоставить пользователям возможность отслеживать свое питание и физическую активность, а также получать рекомендации по упражнениям в зависимости от целей и уровня подготовки.

Функционал приложения
1. Калькулятор питания
   - Ввод данных о весе, росте, возрасте и уровне физической активности пользователя.
   - Расчет калорийной нормы (базовый обмен веществ, уровень активности).
   - Возможность выбора целей: похудение, поддержание веса, набор массы.
   - База данных продуктов с указанием их калорийности и пищевой ценности.
   - Возможность добавления пользовательских продуктов.
   - Функция ведения дневника питания с возможностью редактирования и удаления записей.
   - Генерация отчетов о потреблении калорий за день/неделю.
2. База данных упражнений
   - Классификация упражнений по группам мышц (грудные, спинные, ног, плечевых, рук и т.д.).
   - Доступность фильтров для поиска упражнений по уровням сложности (начальный, средний, продвинутый).
   - Подробные описания к каждому упражнению, включая правильную технику выполнения, количество повторений и подходов.
- Возможность добавления пользовательских упражнений.
 
3. Персонализированные тренировки
   - Генерация программ тренировок на основе выбранной цели и уровня подготовки.
   - Функция планирования тренировок на неделю (с возможностью редактирования).
   - Возможность отслеживания прогресса с графиками и статистикой.
4. Профиль пользователя
   - Личный кабинет для хранения информации о пользователе (фото, контактные данные, история тренировок).
5. Интерфейс и дизайн
   - Удобный и интуитивно понятный интерфейс, соответствующий современным трендам.
   - Адаптивный дизайн для различных устройств (смартфоны, планшеты).

- Платформы: iOS и Android.
- Язык программирования: c++
